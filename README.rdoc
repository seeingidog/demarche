= Demarche

Simple but powerful help desk with bug tracker features.

== Use

Configure in .env:

IMAP_USERNAME=support@ruby-code.com
IMAP_HOST=ruby-code.com
IMAP_PASSWORD=asdfsajdf
MAILBOX_NAME=support

  foreman start
